# Code Of Conduct

## Our standard

Contributors are expected to be respectful, constructive, and professional.

This project welcomes:

- students
- mentors
- maintainers
- first-time contributors

## Unacceptable behavior

- harassment
- discrimination
- personal attacks
- doxxing
- intentionally disruptive behavior

## Enforcement

Project maintainers may remove comments, contributions, or participants that violate these expectations.
